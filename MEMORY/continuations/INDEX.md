## How to Use

In a new session, say:

> "Check my continuations" or "What was I working on?"

> "Continue financial setup" or "Pick up where we left off on finances"

## Continuation Format

Each continuation file contains:
1. **Context** - What led to this point
2. **Decision Points** - Any choices needed
3. **Tasks** - Checklist of what to do
4. **Start Here** - Where to begin
